# 🎯 Tracking Platform — Роботизированная платформа слежения

Экспериментальная роботизированная поворотная платформа для автоматического слежения и предсказания траектории движущихся воздушных объектов (модели самолётов, дроны и т.п.). Учебно-исследовательский DIY-проект.

---

## 📋 Состав системы

| Компонент | Модель | Назначение |
|-----------|--------|------------|
| Компьютер | Raspberry Pi 5 8 ГБ | Управление всей системой |
| AI-ускоритель | AI HAT+ 2 (Hailo-10H 40 TOPS) | Детекция объектов YOLO |
| Камера | Arducam 5MP 1080p PTZ | Захват видео + оптический зум |
| Дальномер | Meskernel TS1224 (UART) | Измерение дистанции лазером |
| Моторы (2 шт.) | Cubemars AK64-80 + драйверы | Поворот платформы (yaw/pitch) |
| CAN-адаптер | Waveshare CAN HAT | Связь с моторами по CAN |
| Джойстик | Logitech X56 HOTAS (правая ручка) | Ручное управление |
| Монитор | 9" автомобильный HDMI | Вывод видео с HUD |

## ⚡ Возможности

### Два режима работы

| Режим | Описание |
|-------|----------|
| **РУЧНОЙ** | Полное управление платформой через джойстик. Стик → скорость поворота. |
| **АВТО** | Камера автоматически удерживает цель в центре кадра через PID-регулятор. |

Переключение одной кнопкой на джойстике.

### Детекция и трекинг
- **YOLO** на Hailo-10H NPU (40 TOPS) — детекция воздушных объектов в реальном времени
- **IoU-трекер** (упрощённый BoT-SORT) — устойчивое отслеживание между кадрами
- Захват/сброс конкретной цели кнопкой

### Баллистический калькулятор
- Расчёт времени полёта снаряда (ToF) с учётом drag и гравитации
- Итеративный расчёт точки перехвата (lead point)
- Учёт атмосферных условий (высота, температура, давление)
- Предсказание траектории цели на основе истории наблюдений

### HUD-прицел (на русском)
- Прицельный крест с мил-дотами
- Уголки bounding box цели (стиль HUD)
- Точка перехвата с линией от цели
- Информационная панель: дистанция, скорость, ToF, зум, углы
- Компас (азимут)
- Статус режима и захвата цели
- Мигающее предупреждение E-STOP

### Управление джойстиком
| Элемент | Действие |
|---------|----------|
| Стик (X/Y) | Наведение платформы |
| Колёсико | Оптический зум камеры |
| Триггер | Замер дальности лазером |
| Кнопка 3 | Захват / сброс цели |
| Кнопка 5 | Переключение РУЧНОЙ ↔ АВТО |
| Кнопка 1 | АВАРИЙНАЯ ОСТАНОВКА |
| Кнопка 2 | Возврат в центр |

## 🚀 Установка

### 1. Клонирование

```bash
git clone https://github.com/your-repo/tracking_platform.git
cd tracking_platform
```

### 2. Установка (на Raspberry Pi 5)

```bash
bash install.sh
```

Скрипт автоматически:
1. Установит системные зависимости
2. Создаст виртуальное окружение Python
3. Установит Python-библиотеки
4. Настроит CAN-интерфейс
5. Создаст скрипты запуска

### 3. Перезагрузка (если CAN настроен впервые)

```bash
sudo reboot
```

### 4. Модель YOLO

Скачайте модель и поместите в `models/`:
- Для Hailo: `models/yolov8n.hef`
- Для Ultralytics: `models/yolov8n.pt`

## ⚙️ Настройка (config/settings.yaml)

Все параметры системы в одном YAML-файле:

- **system** — частота цикла, режим по умолчанию, логирование
- **can** — интерфейс и скорость CAN-шины
- **motors** — CAN ID, лимиты углов, скорости, PID-gains
- **camera** — разрешение, FPS, зум, поле зрения
- **rangefinder** — UART-порт, скорость, пределы дальности
- **detection** — модель YOLO, порог уверенности, классы
- **tracker** — параметры трекера (max_age, min_hits, IoU)
- **joystick** — назначение осей и кнопок, чувствительность
- **autotrack** — PID-коэффициенты автослежения
- **ballistics** — скорость снаряда, drag, атмосфера
- **hud** — цвета, размеры, элементы прицела

## 🛡️ Безопасность

- **E-STOP** (кнопка 1) — мгновенное отключение моторов
- **Ctrl+C** — корректное завершение с отключением всего оборудования
- **Лимиты углов** — программные ограничения поворота
- **Лимиты тока** — защита моторов
- **State Machine** — безопасные переходы между режимами
